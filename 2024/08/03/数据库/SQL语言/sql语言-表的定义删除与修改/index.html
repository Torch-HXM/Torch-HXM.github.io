创建表

前置内容

  1. 什么是主键

    • 主键可以唯一的标识一行，主键可以是多个键，主键具有唯一性
    • PRIMARY KEY 是主键的关键字
    • <列名> 类型 PRIMARY KEY， 这是在创建表时指定单主键的方式
    • PRIMARY KEY(<列名1>, <列名2>) 该语句通常位于建表语句的最后，用来将多个列指定为主键

  2. 表的外联

    • 对表的键进行外联，将其连接到另一个表的列上，让该表的该键列中的值都在其外链的列中
    • foreign key (<键值1>) references <表名2>(<键名2>)

  3. 数据类型

    数据类型 含义
    char(n) 定长字符串
    vachar(n) 最大长度为n的变长字符串
    CLOB 字符串大对象
    BLOB 二进制大对象
    INT 四字节整形
    SMALLINT 二字节整形
    BIGINT 八字节整形
    NUMERIC(p, d) 定点数，小数点前p位整数，小数点后d位小数
    DEC(p, d) 与NUMERIC(p, d)相同
    REAL 取决于机器的单精度浮点数
    DOUBLE PRECISION 取决于机器的双精度浮点数
    FLOAT(n) 可选精度的浮点数，精度至少为n位
    BOOLEAN 逻辑量
    DATE 年月日格式为YYYY-MM-DD
    TIME 时分秒格式为HH:MM:SS
    TIMESTAMP 时间戳类型
    INTERVAL 时间间隔类型
