    Linux的基础符号知识

    重定向符号

    上面两个命令是不一样的，> 代表清空原文件并写入；>> 代表追加。

    正则表达式

    • ^：字符串的开头
    • $：字符串的结束

    常用 Linxu 命令

    cat

    常用功能：

    • 查看文件内容：cat file.txt
    • 合并文件：cat file1.txt file2.txt > file3.txt
    • 创建文件并写入内容：cat >file.txt，回车后即可写入内容，Ctrl+C 退出写入模式。注意，只有当输入行被回车键入时才会写入文件。
    • 追加内容到文件结尾：cat >>file.txt <<EOF，回车后即可写入内容，内容完成后回车，输入 EOF 再次回车，完成对文件的追加。
    • 清空文件内容：cat /dev/null >file.txt
    • 向文件中追加

    echo

    可以将文字显示在命令行中，也可以通过重定向符号写入文件。

    grep

    一些常用命令：

    • -v：显示不匹配的行，显示不包含匹配文本的所有行
    • -n：显示匹配的行及行号
    • -i：不区分大小写
    • -c：统计匹配的行数，不是匹配的次数
    • -E：使用扩展的 egrep 命令，支持正则表达式
    • –color=auto：过滤指定字符串并添加颜色
    • -w：以单词为单位进行过滤
    • -o：只输出匹配内容

    sed 流编辑器

    选项：

    • -n：取消默认输出
    • -i：将修改写入到文件
    • -e：允许多次编辑
    • -r：支持扩展正则表达式

    命令符：

    • a：追加文本
    • d：删除匹配行的文本
    • i：在指定行前插入文本
    • p：打印匹配行内容
    • g：使替换具有全局性
    • s：替换字符串

    测试用例：

    • 打印文件内容
    • 打印指定行内容（范围）
    • 打印包含指定字符串的行
    • 删除包含指定字符串的行，但不写入文件
    • 删除包含指定字符串的行，且写入文件
    • 替换字符串
    • 多次修改
    • 向指定行后面插入文本
    • 向指定行的上面插入文本

    awk

    option 参数选项

    • -F：指定字段分隔符
    • -v：定义或修改一个 awk 内部变量

    常见变量

    • S0：当前整行记录
    • Sn：当前记录的第 n 列，由 FS 或 -F 指定分隔符
    • NF：当前记录中列的个数
    • S(NF-n)：倒数第 n+1 列
    • NR：行号

    测试用例

    • 打印指定行的内容
    • 找出包含指定字符串的行

    crontab

    用以重复的执行命令

    • -u：用来指定用户，缺省时默认是执行命令的用户。只有 root 用户可以指定其他用户。
    • -e：执行文字编辑器来设定时程表
    • -r：删除目前的时程表
    • -l：列出当前的时程表

    变量和类型

    普通变量

    a=5
    定义了一个普通变量，只能在当前shell中访问。

    环境变量

    export a=5
    定义了一个环境变量，shell 和它的子 shell 都可以访问环境变量。

    引号

    双引号和无引号都会解析引用的变量；而单引号不会解析。
